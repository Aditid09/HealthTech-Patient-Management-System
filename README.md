# Health-Tech Patient Management System

A professional Healthcare Database System built for TCET Mumbai B.Voc Data Analytics project.

## 🚀 Features
- **Relational Database:** 3NF Normalized MySQL schema.
- **Data Integrity:** UNIQUE, CHECK, and FOREIGN KEY constraints.
- **Performance:** SQL Indexing for fast patient search.
- **Dashboard:** Real-time metrics using Streamlit.

## 🛠️ Tech Stack
- **Frontend:** Streamlit (Python)
- **Backend:** Python (MySQL Connector)
- **Database:** MySQL Server

## 📖 Database Schema
The system uses two main tables:
1. `Patients`: Stores personal details.
2. `MedicalRecords`: Stores clinical history linked via Foreign Key.
